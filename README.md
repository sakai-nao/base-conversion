# 基数変換学習アプリ

## 概要
2進数、10進数、16進数の相互変換と2進数の四則演算を学習できるWebアプリケーションです。

## 機能
- **基数変換練習**
  - 2進数 ⇄ 10進数
  - 2進数 ⇄ 16進数  
  - 10進数 ⇄ 16進数
- **2進数の計算**
  - 加算
  - 減算
- **学習支援機能**
  - 選択式問題
  - 即座のフィードバック
  - 正解率の表示

## 使い方
1. メニューから学習したい変換方式を選択
2. 問題が表示されたら、正しい答えを選択
3. 「解答」ボタンで答え合わせ
4. 「次の問題」で続行
